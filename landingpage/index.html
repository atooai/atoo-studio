Capabilities

Everything agents need.
Nothing they don't.

A complete development environment purpose-built for AI agent workflows — from proxy interception to live browser preview.

🔀
MITM Proxy & API Orchestration
Intercept and orchestrate all Anthropic API traffic through an intelligent HTTPS proxy. Observe token usage in real-time, approve or deny tool calls, and manage permission modes — all without modifying your agent code.
HTTPS Interception Token Tracking Tool Approval Permission Modes Real-time Streaming
🖥️
Live Browser Preview
Docker-powered Chrome + Xvfb container with WebSocket screen streaming, mouse/keyboard control, viewport emulation, and DevTools proxy.
📡
SSH & Remote Execution
Manage connections with password or key auth, browse remote files, run commands, and spawn terminals over SSH channels.
🔌
MCP Server
Built-in Model Context Protocol server with tools for TCP service reporting, TLS certificate generation, serial device access, and session search.
📂
File Tracking & Change Store
Content-addressed object store with LD_PRELOAD hooks and inotify watchers. Every create, modify, delete, and rename tracked per session.
🔧
Serial Device Support
Full CUSE device support with DTR/RTS modem control signals on Linux, portable PTY fallback, and browser-based port selection.
🌿
Git & Worktree Aware
Automatic git worktree detection, commit history, diffs, branch operations, and project-scoped session filtering that understands your repo structure.
