# Contributing to Atoo Studio

Thanks for your interest in Atoo Studio!

## Code contributions are not accepted yet

Atoo Studio is in **early alpha** and the architecture is evolving quickly. External pull requests would likely conflict with ongoing changes, so **code contributions are not being accepted at this stage**.

This will change as the project stabilizes.

## How you can help

Bug reports, feature ideas, and workflow pain points are very welcome. Please [open an issue](https://github.com/atooai/atoo-studio/issues) with:

- **Bug reports:** Steps to reproduce, expected vs. actual behavior, your OS and Node.js version.
- **Feature requests:** Describe the feature and why it would be useful.
- **Workflow feedback:** Share how you use coding agents and what friction points you run into.
