# File Explorer & Editor

Atoo Studio includes a file explorer with a multi-mode editor for working with project files.

## File Tree

- Navigate project files with a tree view
- Show or hide hidden files
- Drag-and-drop between the browser and your OS file manager
- Works with both local and remote (SSH) projects

## Editor Modes

### Source View

Syntax-highlighted code editing powered by Monaco Editor (the same editor used in VS Code). Supports all major programming languages.

### Diff View

Side-by-side comparison of file changes. Useful for reviewing modifications before committing.

### Rendered View

Preview rendered output for Markdown and HTML files directly in the editor panel.

### Hex View

A hex editor with search and replace — useful for inspecting binary files or debugging encoding issues.

## Tab Management

- Open multiple files in tabs
- Drag-and-drop tab reordering
- Context menus for common operations
- Agents can open files in the editor via the `open_file` MCP tool
